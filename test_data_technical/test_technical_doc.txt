AD5522 Precision Analog Microcontroller
        
        FEATURES:
        • FI voltage clamps and FV current clamps
        • MEASURE VOLTAGE function
        • VMI voltage output range: ±10V
        • Maximum sampling rate: 1MHz
        • Input impedance: 1MΩ || 10pF
        • Temperature range: -40°C to +85°C
        • Supply voltage: ±15V
        
        APPLICATIONS:
        • Automatic test equipment
        • Precision measurement systems
        • Electronic calibration
